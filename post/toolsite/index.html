网站 工具 推荐

这是一些我平时用到的，非常好用的工具网站。
如果有网站失效了，请联系我去除链接。

  • 必备工具
  • 咨讯阅读
  • 科研必备
  • 教程文档

必备工具

  • https://draw.io
    画图网站，各种流程图、类图、甚至房间的房型图都能画，与 https://www.processon.com/ 类似，但是draw.io免费，导出的图也是高清的图片，非常非常推荐！

  • https://carbon.now.sh
    给别人分享代码，如果直接粘贴给别人，非常难看。但是用这个网站就可以生成好看的代码，可以高亮几乎任何语言，并且有很多主题可供挑选。

  • https://www.latexlive.com
    在线latex公式编辑器，加载速度快，而且功能丰富。非常推荐。

  • https://https://patorjk.com/software/taag/#p=display&f=Graffiti&t=Type Something
    这个网站可以把文本转换成ASCII码的艺术字。可以用在github profile、程序简介等。

咨讯阅读

  • https://sspai.com
    少数派 是专注于数字工具和实用方法的优质内容平台，以少数的专业作者为创作基础，不断的整编输出高质量和系统化的数字消费内容。少数派的slogan 高效工作，品质生活 就代表了它的网站功能，经常在上面阅读，能获得很多科技新知。

科研必备

  • https://readpaper.com
    一个读论文的工具网站，可以找到一些论文的pdf版本，并且利用网站提供的阅读工具翻译、批注等。但是这个是一个新网站，不知道能够运营到什么时候。

  • https://copytranslator.github.io
    一款开源翻译软件，复制即翻译的外文辅助阅读翻译解决方案。

复制翻译
只需复制文本到剪贴板，下一秒即可查看翻译结果，享受所见即所得的快感。
优化翻译
解决多余的断句和换行带来的乱码问题，翻译结果更符合阅读习惯。
拖拽复制
无限接近划译的系统级开源实现，拖拽选中即可复制翻译。

教程文档

  • https://www.runoob.com/markdown
    markdown 语言的教程，忘了markdown怎么写，就打开这个现场科研。

  • https://how2j.cn
    学习Java技术栈的好网站，可操作性非常强。

  • https://chinese.freecodecamp.org/
    免费学习编程构建项目

  • https://fullstackopen.com/zh/
    学习全栈的很好的网站，一站式学习 React, Redux, Node.js, MongoDB, GraphQL 以及 TypeScript！这门课程会向你介绍基于 JavaScript 的现代 Web 编程技术。重点是利用 ReactJS 搭配Node.js开发的REST API，来搭建单页面应用程序（SPA：Single Page Application）
